# FluxPoint

FluxPoint is a motion-heavy landing page I built to pitch an event-driven methane abatement platform for critical infrastructure.

## What this product communicates

I built this as a strategic narrative site: define the methane monitoring gap, explain the architecture, and present FluxPoint as the bridge between satellite visibility and on-ground action.

## Core features

- Story-driven one-page pitch structure
- Scroll-reactive sections and highlight animations
- Architecture and case-study storytelling blocks
- CFD visual module (`CFDAnimation`)

## Stack

- React (Vite)
- Tailwind CSS
- Framer Motion
- D3

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
